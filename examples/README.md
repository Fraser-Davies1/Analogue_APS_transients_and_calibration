# Examples

All examples in this directory are sourced from `pytestlab/examples`, organized by topic.

Folders:
- `getting-started/`
- `bench/`
- `sweep/`
- `measurement/`
- `plotting/`
- `database/`
- `compliance/`
- `testing/`
- `parallel/`
- `replay_mode/`
- `scripts/`


The `smartbench.yaml` file contains the default for instruments available in the lab.
